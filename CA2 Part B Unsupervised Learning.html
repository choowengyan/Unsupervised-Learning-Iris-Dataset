Find the optimum clusters (Hierarchy)¶

Before creating the model, optimum number of clusters have to be found so that we will get to know what number to put for n_clusters by using cluster hierarchy (dendrogram)

In [8]:
from scipy.cluster.hierarchy import dendrogram, linkage

merg = linkage(features,method="ward")

plt.figure(figsize=(18,6))
dendrogram(merg, leaf_rotation=90)
plt.xlabel("data points")
plt.ylabel("euclidian distance")

plt.suptitle("DENDROGRAM",fontsize=18)
plt.show()

From the hierarchy chart above, it shows that the longest vertical line without any perpendicular matching lines (which is known as the Euclidian distance) is about 3.

This shows that there 3 interceptions which represents the ideal amount of labels in Iris Dataset are divided into 3 clusters.

K Means Clustering¶

The Elbow Method/ Screeplot¶

Find the best value for k in K-means classification¶

Before creating the K-Means model, optimum number of clusters have to be found so that we will get to know what number to put for n_clusters by using screeplot (also known as the Elbow Method)

In [8]:
from sklearn.cluster import KMeans
wcss = []

for k in range(1,15):
    kmeans = KMeans(n_clusters=k)
    kmeans.fit(features)
    wcss.append(kmeans.inertia_)


plt.figure(figsize=(8,5))
plt.title("Screeplot", fontsize=18)
plt.plot(range(1,15),wcss,"-o")
# plt.grid(True)
plt.xlabel("Amount of Clusters",fontsize=14)
plt.ylabel("Inertia",fontsize=14)
plt.xticks(range(1,15))
plt.tight_layout()
plt.show()

The graph on the above shows the screeplot to show the best number to put for K-Means Clustering.

Based on the elbow rule, the optimum clusters is where the elbow occurs and it's when the within cluster sum of squares (WCSS) does not decrease significantly every iteration.

Hence, it seems like 3 or 2 will be the best value for clusters.

Silhouette Score¶

In [16]:
# Clustering silhouette_score
from sklearn.metrics import silhouette_score

for n_cluster in range(2, 11):
    kmeans = KMeans(n_clusters=n_cluster).fit(X)
    label = kmeans.labels_
    sil_coeff = silhouette_score(X, label, metric='euclidean')
    print("For n_clusters={}, The Silhouette Coefficient is {}".format(
            n_cluster, sil_coeff))

The silhouette coefficient method measures the quality of clustering by determining how well a data point lies within its cluster. A high average silhouette indicates good clustering.

After the iterations, it shows that K-Means Clustering with 2 or 3 n_clusters have the highest Silhouette Score which has the closest score to 1.

Hence, I will use 2 and 3 n_clusters to predict the iris species and see which clusters gets a higher rand score.
